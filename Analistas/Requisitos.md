# Requisitos del Sistema

A continuación se detallan los requisitos de dominio identificados para el caso de estudio.

| Número | Nombre           | Descripción                                                               |
| ------ | ---------------- | ------------------------------------------------------------------------- |
| RD1    | Software_App_Web | El sistema debe implementarse como una aplicación web. La comunicación y transferencia de datos entre los componentes del cliente (el módulo de visualización ) y el servidor (el módulo inteligente o de suscripción ) debe realizarse obligatoriamente usando los protocolos estándar HTTP/REST. |
